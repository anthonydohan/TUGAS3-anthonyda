Belajar CSS

Apa itu CSS?

Cascading Style Sheet merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemrograman.

Bagaimana menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. Inline. Metode ini digunakan dengan cara memasukkan file css ke dalam tag HTML secara langsung.
  2. Internal. Metode ini menggunakan CSS dengan tag style pada file html.
  3. External. Metode ini digunakan dengan cara memisahkan file html dan juga file css .
